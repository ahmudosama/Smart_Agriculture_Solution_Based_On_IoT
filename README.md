# Smart Agriculture Solution based on IoT

## Project Overview
This project presents a comprehensive IoT-based smart agriculture solution designed to enhance crop management and optimize resource use. It leverages various sensors, a Raspberry Pi 3 Model B+, and AI for real-time pest detection, aiming to improve productivity and sustainability in agriculture.

## Features
- **Sensor Integration**: Monitors soil moisture, temperature, humidity, rainfall, light intensity, and motion.
- **Water Pump Control**: Automated irrigation using a 12V DC water pump controlled via a relay.
- **AI Pest Detection**: Real-time pest detection using a YOLO model deployed on a Raspberry Pi.
- **Remote Monitoring**: Uses Blynk for IoT environment control and monitoring.
- **Power Management**: Utilizes solar power and a power bank for energy efficiency.

## Components
- **Raspberry Pi 3 Model B+**
- **Sensors**:
  - Soil Moisture Sensor
  - DHT11 Temperature and Humidity Sensor
  - Rain Sensor
  - LDR Light Sensor
  - PIR Motion Sensor
- **Water Pump**: 12V DC water pump with relay
- **MCP3008**: Analog to digital converter
- **Power Supply**: Power bank (10,000 mAh), Solar panel (20W, 18V), DC to DC converter (5V, 2A)
